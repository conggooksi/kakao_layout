부트스트랩 레이아웃

레이아웃 관련태그를 하나로 묶기 위해 사용할 수 있는 클래스는 container(화면크기 고정: 1170px), container-fluid(전체화면크기) 등이 있음

반응형 웹에 적용할 일반적인 중단점breakpoint은 576px(스마트폰), 768px(태블릿), 992px(일반적인 데스크탑), 1200px(넓은화면)임

그리드 레이아웃

웹 페이지의 구성요소들을 특정 너비로 나누어진 열을 기준으로 배치하는 것을 의미

부트스트랩은 12열 그리드 시스템을 이용함

레이아웃을 구성하는 방법

  1. container/container-fluid 선택
  2. row 클래스로 행을 배치
  3. col-*-n 클래스로 컬럼을 배치
  4. 컬럼 안에 컨텐츠를 배치
